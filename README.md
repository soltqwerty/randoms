## Projekt leírás
Ez a projekt egy **Cisco Packet Tracer** prototípus, amely a hálózati alapelvek gyakorlati alkalmazását mutatja be. A cél egy működő, átlátható hálózati modell létrehozása, amelyet a dokumentáció és a prezentáció is részletesen ismertet.

## Fájlok
| Fájl | Leírás |
|------|--------|
| `randoms_finished.pkt` | Packet Tracer prototípus |
| `randoms_dokumentáció.docx` | Részletes dokumentáció a hálózatról |
| `randoms_prezentáció_hu.pptx` | Prezentáció a projekt bemutatásához(magyar nyelven) |
| `randoms_prezentáció_en.pptx` | Prezentáció a projekt bemutatásához(angol nyelven) |
| `randoms_tesztelés.xlsx` | Tesztelési séma a Packet Tracer prototípushoz |

## Használat
1. Nyisd meg a Packet Tracer fájlt (`randoms_finished.pkt`) a Cisco Packet Tracer programmal.
2. Tanulmányozd a hálózati konfigurációkat és kapcsolódásokat.
3. A dokumentáció (`randoms_dokumentáció.docx`) részletes útmutatást nyújt a hálózat működéséről.
4. A prezentáció (`randoms_prezentáció_hu.pptx`) a projekt bemutatásához használható.

## Megjegyzés
A projekt célja, hogy bemutassa a hálózati eszközök konfigurációját és a kapcsolódási logikát.
